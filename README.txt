GUIDE TO USE OUR CAFE ANALOG WEBSITE

Step 1: Download the zip file
Step 2: Extract the folder CafeAnalog
Step 3: Make sure you have Node.js installed
Step 4: In CMD, run the command npm install http-server -g
Step 5: Find the folder CafeAnalog in your terminal
Step 6: Run the command http-server
Step 7: Type localhost:8080/Frontpage.html in your browser
Step 8: Enjoy


IF YOU HAVE INTELLIJ

Step 1: Download the zip file
Step 2: Extract the folder uewp2022-start
Step 3: Open the folder in intellij
Step 4: Use intellij's host feature
Step 5: Enjoy
